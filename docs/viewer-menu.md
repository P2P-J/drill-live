### 💬 채팅 명령어 (무료!)

> `!` 있어도 되고 없어도 됩니다. `bomb` / `!bomb` 둘 다 동작.

| 명령어 | 효과 |
|--------|------|
| `bomb` | 작은 TNT 한 개 💣 |
| `fast` | 드릴 속도 ×1.5 (10초) ⚡ |
| `wood` | 나무 드릴로 변경 (3초) 🟫 |
| `stone` | 돌 드릴로 변경 (3초) ⬜ |
| `iron` | 철 드릴로 변경 (3초) ⚙️ |
| `gold` | 금 드릴로 변경 (3초) 🟨 |
| `diamond` | 다이아 드릴로 변경 (3초) 💎 |
| `range` | 드릴 범위 +1 (드릴이 커짐!) — 5초 |
| `engine` | 이동 속도 +1 — 5초 |

> `fast`는 전체 채널 쿨다운 10초.
> 드릴 컨셉(wood~diamond)은 무료 — 같은 컨셉 또 치면 시간 리프레시.

### ❤️ 공짜인데 화면에 닉네임이 떠요!

- 👍 **좋아요(LIKE)** → 좌측 하단 LIKE 피드에 💗 닉네임 표시 + 작은 TNT 떨어짐
- ⭐ **구독(SUBSCRIBE)** → 화면 중앙 큰 배너 + **드릴 아래 10줄 전체가 해당 바이옴 특수 광물로 변환!**
- 💎 **멤버십 가입** → 다이아 드롭 + 닉네임 큰 배너
