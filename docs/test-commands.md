# 🎮 테스트 키보드 명령어 (스트리머 전용)

방송 중 실제 후원 없이 트리거를 테스트할 때 사용. 게임 화면(`http://localhost:3000`)에 포커스를 두고 키를 누르면 실제 후원이 들어온 것처럼 동일하게 동작.

> ⚠️ 시청자에게는 화면 결과(폭발/광물 스폰/announcement)만 보이고, 어떤 키를 눌렀는지는 표시되지 않음. 매번 17명 가상 후원자 풀에서 무작위로 이름이 뽑혀 표시됨.

---

## 💣 폭발 계열 (TNT 떨어지고 1초 치지직 후 폭발)

| 키 | 트리거 | 가격 | 반경 (타일) | 비고 |
|---|---|---|---|---|
| `1` | BOMB | $1 | 1.6 | 기본 폭탄 |
| `2` | ULTRA BOMB | $3 | 2.6 | 더 큰 폭탄 |
| `3` | MEGA BLAST | $5 | 3.6 | 다이너마이트 (큰 반경) |
| `4` | NUKE | $20 | 6.0 | 화면 절반 + 강한 흔들림 + 벽도 깸 |

폭탄은 **드릴 크기와 무관하게 항상 첫 번째 살아있는 타일(땅)까지 떨어진** 후, **1초간 도화선 치지직** 효과 후 폭발. 작은 폭탄(반경 < 3)은 회색 벽은 못 깨고, NUKE/MEGA BLAST는 벽도 깸.

## ⚡ 드릴 버프 계열 (지속 효과)

| 키 | 트리거 | 가격 | 효과 | 지속 |
|---|---|---|---|---|
| `5` | DRILL UP | $2 | 드릴 파워 + 이동속도 ×1.5 | **30초** |
| `6` | TURBO | $5 | 드릴 파워 + 이동속도 ×3.0 | **15초** |
| `7` | OVERDRIVE | $10 | 드릴 파워 + 이동속도 ×5.0 | **20초** |
| `W` | !fast (채팅) | 무료 | 드릴 파워 ×1.5 | **10초** (채널 쿨다운 30초) |
| `SPACE` | RANGE UP | SC | 드릴 폭 +2단계 (드릴 커짐) | **10초** |

> 같은 효과 중복 시 PRD 룰대로 **지속시간 갱신** (중첩 아님). 우상단에 잔여시간 막대 표시.
> `!fast`는 채팅 트리거로, 누가 누르든 **채널 전체 30초 쿨다운**이 걸려서 그 안에 다시 `W`를 눌러도 무시됨 (spec 5-1).

## ❤️ 좋아요 (YouTube LIKE)

| 키 | 트리거 | 효과 |
|---|---|---|
| `L` | LIKE | **가장 작은 TNT**(반경 1.2)가 드릴 아래 떨어짐 → 3초 sizzle → 폭발 |

- sizzle 3초 동안 추가로 `L`을 누르면 **같은 TNT에 이름이 누적**됨 (TNT 위에 노란 글씨로 최근 5명까지 표시).
- 폭발 후 다음 LIKE는 새 TNT로 처음부터 시작.
- 실제 라이브에서는 좋아요를 누른 사람들을 묶어서 보여주려는 의도 (좋아요 1개당 폭탄 1개면 화면이 폭탄으로 가득 차서 게임 망가짐).

## 👨‍✈️ 스트리머 전용 (채팅에서 owner/moderator만 사용 가능)

| 키 | 트리거 | 효과 | 채팅 명령어 |
|---|---|---|---|
| `F` | JACKPOT | 현재 화면 빈 흙 타일의 40%를 다이아몬드로 변환 + 금색 플래시 | `!jackpot` |
| `Z` | RESET | 모든 청크 destroy → 재생성. 드릴 위치/골드/인벤은 유지. 흰색 플래시. | `!reset` |
| `B` | BOSS_SPAWN | 다음 보스 강제 소환 (현재 깊이 무시) | `!boss_spawn` |

## 🛠️ 개발/디버그 단축키 (시청자에게 표시 안 됨)

| 키 | 효과 |
|---|---|
| `G` | 골드 +10,000 즉시 추가 (업그레이드 빠르게 사기) |
| `D` | 깊이 +1,000km 점프 (바이옴 전환 빠른 확인) |
| `R` | 드릴 폭 단계 토글 (1→2→3→1) — UI 버프 안 거치고 즉시 변경 |
| `P` | 드릴 파워 단계 토글 (1→5→1) — 드릴 색상도 흰→돌→철→금→다이아 순으로 변함 |
| `N` | 현재 활성 보스 즉시 처치 |
| `;` | 사운드 mute/unmute 토글 |

> `D` + Shift = +50,000km 점프 (Crystal Cave 진입 등 빠른 확인용)

## 💡 사용 시나리오

### 방송 시작 전 점검 (1분)
```
G              # 골드 충분히 확보
1, 2, 3, 4     # 폭탄 4종 시각 확인
8, 9, 0        # 광물 스폰 확인
6              # TURBO로 잠시 빠르게
D, D, D        # 깊이 점프해서 다른 바이옴 톤 확인
```

### 시청자 반응 확인 (방송 중)
- 채팅이 한동안 조용하면 `1`을 눌러서 BOMB 폭발 시뮬레이션 → "어 누가 후원했네?" 분위기 유도

### 보스 시스템 테스트
- `B` 한 번 → 현재 위치에서 Mega Mole 즉시 등장 (아레나 형성). `1, 2, 3, 4`로 보스 체력 깎기 → `N`으로 즉시 처치.
- 깊이 기반 자연 등장 테스트: `D` 9번 점프해 9,000km 부근 도달 → 자동 소환 확인.

### 좋아요 누적 테스트
- `L` → 1번째 TNT 등장 + sizzle 시작. 3초 안에 `L`을 3~4번 더 → TNT 위에 노란 이름 누적 표시 → 한 번에 폭발.

## 🔌 OBS 설정 시 주의

1. **브라우저 소스 추가**: `http://localhost:3000`, 너비 1080, 높이 1920
2. 브라우저 소스를 **클릭해서 포커스**시키지 않으면 키 입력 안 받음
3. 방송 중에는 게임 창을 별도로 열어두고 거기에 포커스 → 키 입력 → OBS는 다른 창의 결과를 캡처 (브라우저 소스가 클릭 인터랙션 받으면 자동 포커스됨)
4. OBS Studio → Sources → 우클릭 → 'Interact'로 키 입력 가능 (방송 모드에서 사용)

## 🚀 Phase 4 이후 (실제 슈퍼챗 연결)

위 키보드 시뮬레이션이 그대로 동작 + 추가로:
- Streamer.bot이 슈퍼챗 감지 → WebSocket으로 게임에 신호 → `triggerSystem.fire('BOMB', donorName)` 자동 호출
- youtube-chat이 채팅 폴링 → `!fast` 같은 채팅 트리거 자동 발동
- 즉, **테스트 키보드 코드는 그대로 살리고**, 실제 후원 시 자동으로 같은 함수가 호출됨
